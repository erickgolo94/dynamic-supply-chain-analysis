Dynamic Supply Chain Analysis
Project Overview

    Description: A comprehensive end-to-end Exploratory Data Analysis (EDA) and dynamic dashboarding project analyzing supply chain inventory, shipping costs, supplier delay probabilities, and risk classifications across multiple international regions.

    Author: Erick Andos Golo - Junior Data Analyst

    Tech Stack: Python (Pandas, NumPy), Tableau

Key Features & Workflow

    Python Data Processing: Performs rigorous data cleaning, handling of anomalies, and exploratory data analysis on a massive dataset of approximately 113,000 records using a Jupyter Notebook (Supply_Chain_Inventory_Optimizer_EDA.ipynb).

    Tableau Visualization: Features interactive metrics breaking down top shipping costs by supplier ID, risk classifications, and delay probabilities by country (such as Norway, Greece, Philippines, and Algeria).
